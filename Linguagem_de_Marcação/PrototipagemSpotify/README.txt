🎵 Spotify UI Clone - Figma

Este projeto é um protótipo de interface do Spotify desenvolvido no Figma, com foco em recriar a experiência visual do aplicativo, desde a tela inicial até a navegação por músicas e playlists.

📱 Telas Desenvolvidas

Splash Screen

Tela inicial com o logotipo do Spotify em destaque.

Tela de Login

Campos para inserção de e-mail e senha.

Opção "Esqueceu a senha?".

Botão Entrar estilizado em verde característico do Spotify.

Tela Principal (Home)

Barra de filtros: All, Music, Podcasts.

Playlists e álbuns sugeridos (ex: Alan Walker, Release Radar, It's New Music Friday).

Cards interativos e rolagem vertical.

Mockup em Dispositivo (iPhone 16 Pro Max)

Visualização do design aplicado em mockups realistas.

🛠️ Ferramentas Utilizadas

Figma (design e prototipagem).

Componentes reutilizáveis para inputs, botões e cards.

Grid e alinhamento responsivo para manter consistência visual.

🚀 Funcionalidades Simuladas

Fluxo de autenticação (login).

Navegação entre playlists e músicas.

Identidade visual baseada no Spotify oficial.

🎯 Objetivo

Este projeto foi desenvolvido para estudo de UI/UX, com foco em:

Treinar boas práticas de design de aplicativos.

Aplicar consistência visual em telas diferentes.

Explorar prototipagem interativa no Figma.
